Aquí encontrarás una ruta simple para consultar recursos del curso (archivos, páginas, enlaces y materiales multimedia) sin perderte en la navegación.

## Pasos rápidos

<Steps>
1. **Entra al curso y ubica la sección.**

	 Localiza la unidad o bloque donde el docente publicó el material.

2. **Identifica el tipo de recurso.**

	 Puede ser archivo descargable, página de lectura, enlace externo o video.

3. **Abre o descarga el recurso.**

	 Revisa que el contenido cargue correctamente y guarda una copia cuando sea necesario.

4. **Revisa instrucciones asociadas.**

	 Algunos recursos incluyen actividades o tareas relacionadas.

5. **Confirma avance.**

	 Si la actividad maneja seguimiento, verifica si quedó marcada como completada.
</Steps>

:::tip[Organización recomendada]
Crea carpetas por materia y unidad para guardar archivos descargados y consultarlos rápido cuando estudies.
:::

## Checklist rápido

- Abrí el recurso correcto dentro de la sección indicada.
- Verifiqué que el contenido cargó completo.
- Revisé si tenía instrucciones o actividad relacionada.
- Guardé el material importante para estudio posterior.

## Problemas comunes

- **El archivo no abre:** intenta descargarlo y abrirlo localmente.
- **El video o enlace externo no carga:** recarga la página o prueba otro navegador.
- **No encuentro el material:** confirma que estás en la unidad correcta y revisa avisos del curso.
